# CoPaw 服务运维

## 启动

```bash
pkill -f "copaw app"
sleep 2
nohup /Users/jiangshengping/miniconda3/bin/copaw app > /tmp/copaw.log 2>&1 & echo 'PID:' $!
```

## 停止

```bash
pkill -f "copaw app"
```

## 重启

```bash
pkill -f "copaw app"
sleep 2
nohup /Users/jiangshengping/miniconda3/bin/copaw app > /tmp/copaw.log 2>&1 & echo 'PID:' $!

/Users/jiangshengping/miniconda3/bin/copaw app

/Users/jiangshengping/miniconda3/bin/copaw app 2>&1 | tee /tmp/copaw.log
```

## 查看日志

```bash
tail -f /tmp/copaw.log
```

---

## 验证服务

```bash
curl -s http://localhost:8088/api/version
# 或
curl -s http://localhost:8088/ | head -2
```

## 验证文件上传

```bash
curl -s -X POST http://localhost:8088/api/files/upload \
  -F "file=@/path/to/your/file.pdf" \
  | python3 -c "import json,sys; d=json.load(sys.stdin); print('✅ 上传成功:', d.get('url',''))"
```

## 验证 Skills

```bash
curl -s "http://localhost:8088/api/skills" | python3 -c "
import json,sys
data = json.load(sys.stdin)
print(f'共 {len(data)} 个 skill：')
for s in data:
    mark = '✅' if 'contract' in s['name'] else '  '
    print(f'  {mark} {s[\"name\"]}')
"
```
